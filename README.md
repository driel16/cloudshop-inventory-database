\# CloudShop Inventory \& Analytics Platform



\## Overview



CloudShop Inventory \& Analytics Platform is a MySQL database project designed to manage products, suppliers, customers, orders, and inventory transactions.



The project demonstrates database design, normalization, SQL querying, and business analytics reporting.



\## Features



\- Product Management

\- Supplier Management

\- Customer Management

\- Inventory Tracking

\- Order Processing

\- Business Reporting



\## Technologies



\- MySQL

\- SQL

\- Relational Database Design



\## Database Tables



1\. Roles

2\. Users

3\. Categories

4\. Suppliers

5\. Products

6\. Customers

7\. Orders

8\. Order\_Items

9\. Inventory\_Transactions
